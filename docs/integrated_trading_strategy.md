## 3. 2단계: 매수 시기 결정 (When to Buy)

### 3.1 전략 A: 추세 추종형 진입

**적용 엔진**: V2, V4, V7

#### RSI 스윗스팟 진입
```
매수 조건:
- RSI 60-75 (모멘텀 최강 구간)
- 또는 RSI 50-60에서 상승 전환
- RSI >80 + 하락 전환 시 → 매수 금지 (-5점)
- RSI <30 → 절대 매수 금지 (낙하하는 칼날)
```

#### 눌림목 거래량 급감 매수
```
조건:
1. 직전 5일 상승 후 조정
2. 조정 시 거래량 전일 대비 -30% 이상 감소
3. 20일선 지지 유지
→ 세력 이탈 없음 확인, 매수
```

#### MACD 골든크로스
| 신호 | 점수 | 의미 |
|------|------|------|
| MACD선이 시그널선 상향 돌파 | +20점 | 강력 매수 |
| MACD 히스토그램 음→양 전환 | +10점 | 모멘텀 전환 |
| MACD > 0 + 상승세 | +5점 | 추세 지속 |

## 4. 3단계: 매도 시기 결정 (When to Sell)

### 4.1 목표가 매도 (Profit Taking)

#### ATR 기반 목표가
| 전략 | 목표가 | 손절가 | R:R |
|------|--------|--------|-----|
| **V6 스윙** | Entry + ATR × 2.0 | Entry - ATR × 1.0 | 2:1 |
| **V7 추세** | Entry + ATR × 1.5 | Entry - ATR × 0.8 | 1.875:1 |
| **V8 반등** | Entry + ATR × 1.5 | Entry - ATR × 0.8 | 1.875:1 |

```python
# ATR 계산 (14일 기준)
ATR = 14일간의 True Range 평균
True Range = max(High-Low, |High-PrevClose|, |Low-PrevClose|)

# 예시: 삼성전자 현재가 80,000원, ATR = 1,500원
목표가 = 80,000 + (1,500 × 1.5) = 82,250원
손절가 = 80,000 - (1,500 × 0.8) = 78,800원
```

#### 과매수 구간 분할 매도
```
1차 매도 (50%): RSI 70 도달
2차 매도 (30%): RSI 80 도달
3차 매도 (20%): RSI 80 이탈 또는 목표가 도달
```

## 7. 리스크 관리 원칙

### 7.1 포지션 사이징

```python
# 켈리 공식 변형 (보수적)
position_size = (win_rate × avg_win - (1 - win_rate) × avg_loss) / avg_win
position_size = position_size × 0.5  # 50% 보수적 적용

# 예시: V7 전략
win_rate = 0.60  # 60% 승률
avg_win = 0.015  # 평균 +1.5%
avg_loss = 0.008  # 평균 -0.8%

kelly = (0.60 × 0.015 - 0.40 × 0.008) / 0.015 = 0.387
conservative_kelly = 0.387 × 0.5 = 19.3%

→ 종목당 최대 20% 포지션
```

### 7.2 일일 최대 손실 한도

```
일일 최대 손실: 총 자산의 -3%
→ 도달 시 당일 신규 매수 중단, 기존 포지션만 관리

주간 최대 손실: 총 자산의 -7%
→ 도달 시 주간 신규 매수 중단, 전략 재검토
```

### 7.3 상관관계 리스크

```
동일 섹터 종목 제한:
- 최대 3종목 (같은 테마)
- 예: 2차전지에서 에코프로, 에코프로비엠, 포스코퓨처엠
  → 3종목 이상 진입 금지 (동반 하락 리스크)
```

### 7.4 뉴스 리스크
