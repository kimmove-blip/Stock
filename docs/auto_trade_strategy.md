## 매수 전략

### 1. Early Surge Detection (09:05 ~ 09:30) - 신규

**목적:** 장 초반 급등주 조기 포착 (V2가 낮을 때 진입)

**배경:**
- 7일간 백테스트 결과, 20%+ 급등종목의 09:02 평균 V2는 47 (V2>=80 조건 미충족)
- 기존 전략(V2>=80)으로는 급등주 포착률 0%
- MACD_BULL + MA시그널 조합이 93% 급등주에서 발견

**조건:**
| 항목 | 조건 | 비고 |
|------|------|------|
| MACD_BULL | 필수 | 93% 급등주 보유 |
| MA 시그널 | 필수 | MA_ALIGNED, MA_STEEP, MA_20_STEEP, MA_20_VERY_STEEP 중 하나 |
| 등락률 | 0% ~ 8% | 갭상승 추격 방지 |
| VOLUME 시그널 | 선택 (가점) | VOLUME_EXPLOSION, VOLUME_SURGE 등 |

**기대효과:**
- 20%+ 급등주 포착률: 0% → 33.8%
- 평균 수익률: +1.54% (기존 대비 +123%)

**코드 위치:** `trading/buy_sell_logic.py:should_buy_advanced()`

## 매도 전략

### 1. 손절

| 항목 | 조건 |
|------|------|
| 손절선 | **-5.0%** |
| 적용 계좌 | 김형철, 조은미 동일 |

### 2. 스코어 기반 매도

**매도 조건 (하나라도 충족 시):**
| 조건 | 설명 |
|------|------|
| V4 < 40 | 수급 악화 |
| V2 < 50 AND V4 < 45 | 추세+수급 복합 악화 |

**홀딩 조건 (충족 시 매도 보류):**
| 조건 | 설명 |
|------|------|
| V5 >= 60 | 강한 눌림목 신호 (추가 상승 기대) |
| V4 >= 55 | 수급 양호 |
| V2 >= 60 | 추세 양호 |

### 3. 장마감 정리매매 (14:55 ~ 15:20)

**목적:** 당일 포지션 정리 및 익일 리스크 최소화

**적용 시간:** 14:55 ~ 15:20

#### 정리매도 판단 기준

| V5 스코어 | 조치 | 근거 |
|-----------|------|------|
| V5 >= 70 | **홀딩** | 강한 눌림목 신호, 익일 추가 상승 기대 |
| V5 < 70 | **정리 매도** | 모멘텀 약화, 갭다운 리스크 |

#### 정리매매 흐름도

```
14:55 장마감 정리 시작
    │
    ├─ 손실 종목 (-5% 이하) → 즉시 손절
    │
    ├─ V5 >= 70 → 홀딩 (익일 홀딩 목록)
    │
    └─ V5 < 70 → 정리 매도
          │
          └─ 수익 여부 무관하게 청산
```

#### 홀딩 조건 상세

| 조건 | 값 | 설명 |
|------|-----|------|
| V5 | >= 70 | 눌림목 + 이평선 밀집 + BB 수축 |
| 손익률 | 무관 | 수익/손실 무관하게 V5 기준 적용 |

**홀딩 시 익일 기대수익:** 백테스트 기준 평균 +12.5%

#### 정리매도 조건 상세

| 조건 | 설명 |
|------|------|
| V5 < 70 | 눌림목 신호 약함 |
| 14:55 이후 | 신규 매수 금지 |

#### 예외 처리

| 상황 | 처리 |
|------|------|
| 상한가 종목 | 매도 불가 → 자동 홀딩 |
| 거래 정지 | 매도 불가 → 자동 홀딩 |
| API 오류 | 재시도 3회 후 스킵 |

**코드 위치:** `auto_trader.py:2218-2224`
