# V1~V10 트레이딩 빠른 참조 가이드

## 엔진 선택 의사결정 트리

```
시장 상황 판단
    │
    ├─ KOSPI +1% 이상, 외인 매수 → 강세장
    │       └─ V2(40%) + V4(25%) + V5(20%) + V10(15%)
    │
    ├─ KOSPI -1% 이하, 외인 매도 → 약세장
    │       └─ V8(40%) + V9(30%) + V1(20%) + 현금(10%)
    │
    ├─ KOSPI ±1% 박스권 → 횡보장
    │       └─ V3(35%) + V6(30%) + V9(25%) + 현금(10%)
    │
    └─ 특정 섹터 급등 → 테마장
            └─ V10(50%) + V4(30%) + V2(20%)
```

## 매도 신호 종합표

| 신호 | 조치 | 우선순위 |
|------|------|----------|
| ATR × 1.5 목표가 도달 | 분할 매도 50% | 1순위 |
| RSI 80 이상 + 하락 전환 | 분할 매도 30% | 2순위 |
| ATR × 0.8 손절가 도달 | 전량 매도 | 즉시 |
| 20일선 종가 이탈 | 전량 매도 | 즉시 |
| 보유 3-5일 미반응 | 전량 매도 | 시간손절 |
| 대장주 상승분 70% 캐치업 | V10 익절 | 1순위 |

## 엔진별 핵심 파라미터

### V2 (추세 추종) - 현재 기본값
```
필수: 정배열 (역배열 시 점수=0)
RSI: 60-75 스윗스팟 (+15), <30 매수금지 (-10)
MA20 기울기: ≥1.5% (+15), ≥0.5% (+10)
거래대금: 500억+ (+15), 100억+ (+10)
```

### V9 (갭상승 ML)
```
확률 임계값: ≥70%
거래대금: ≥50억
등락률: -15% ~ +25%
진입: 15:20 장마감
청산: 익일 시가
예상 승률: 61.2%, 순수익 +0.185%/건
```

### V10 (대장주-종속주)
```
상관계수: ≥0.5 (0.7+ 권장)
대장주: +3% 이상 상승
종속주: +2% 이하
캐치업 갭: ≥2%
손절: -3%
시간손절: 3일
목표: 대장주 상승분 70-80% 캐치업
```
